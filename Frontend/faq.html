About the Platform

Achievely is a Steam achievement explorer that lets you search any game and instantly see its full achievement list — including global completion rates, hidden achievements, and rarity tiers.

If you connect your Steam account, Achievely layers your personal unlock history on top: you'll see which achievements you've already earned, your overall completion percentage per game, and a snapshot of your entire library sorted by progress.

Data is sourced from the RAWG video game database for game metadata and directly from the Steam Web API for achievement and profile information.

Steam Connection

Your Steam ID is a unique 17-digit number. Here's how to find it:

  1. Open Steam and click your username in the top right.
  2. Select Account Details.
  3. Your Steam ID is displayed below your username — it looks like 76561198XXXXXXXXX.

Alternatively, visit store.steampowered.com/account while logged in.

If your Steam URL uses a custom name (e.g. steamcommunity.com/id/yourname), use a tool like steamid.io to convert it to the numeric ID.

Steam controls access to your profile data. When your profile is set to Private or Friends only, Steam's API blocks third-party apps — including Achievely — from reading your game list and achievements.

To change this:

  1. Go to Steam → Settings → Privacy.
  2. Set My Profile and Game details to Public.

You can switch it back to Private after exploring — Achievely doesn't cache your data between sessions.

Only publicly visible information is read — the same data anyone could see on your Steam profile page:

  • Display name and avatar
  • Owned games and playtime
  • Achievement unlock status per game
  • Steam level and account creation date

Achievely never accesses your password, payment information, friends list messages, or any private data. We use read-only API keys and cannot perform any actions on your behalf.

Features

Each achievement shows the percentage of Steam players who have unlocked it globally. Achievely groups these into four tiers:

  • Ultra Rare — under 5% of players have it
  • Rare — 5–19%
  • Uncommon — 20–40%
  • Common — above 40%

Some developers mark achievements as "hidden" to avoid spoiling story moments or secret mechanics. Steam normally shows these as question marks.

On Achievely, hidden achievements show their name but keep the description blurred. Click any hidden achievement card to instantly reveal its description — no spoiler by accident.

Click the bookmark icon on any game's detail page to save it. Your bookmarks are stored locally in your browser and accessible from every page via the bookmark icon in the top navigation bar.

On the Achievement Explorer page, the bookmark icon next to the search bar lets you quickly jump to any saved game's achievement list without searching again.

Privacy & Security

Your Steam ID is stored only in your browser's local storage — it never leaves your device unless you make a request to the API. There is no Achievely account, no server-side database of users, and no persistent session.

Clearing your browser's local storage or clicking Disconnect removes everything immediately.

No. Achievely makes API requests to Steam on your behalf to fetch publicly visible data, and that's it. We don't sell data, don't run analytics on user behaviour, and don't use advertising networks.

The only external services involved are:

  • Steam Web API — for achievement and profile data
  • RAWG API — for game metadata and covers
  • Google Fonts — for typography (standard CDN request)

No cookies. Achievely uses the browser's localStorage to remember your Steam ID and display preferences (like bookmarked games) between visits. This is local-only and contains no tracking information.

Troubleshooting

Check these in order:

  1. Profile visibility — go to Steam Settings → Privacy and make sure both My Profile and Game details are set to Public.
  2. Correct Steam ID — make sure you entered the full 17-digit numeric ID, not your vanity URL name.
  3. Game is on Steam — achievement data is only available for Steam games. Games from Epic, GOG, or console platforms won't show achievements.
  4. No achievements in the game — some Steam games (especially older or indie titles) simply don't have achievements implemented by the developer.
Try disconnecting and reconnecting your Steam ID on the Profile page to force a fresh data fetch.

The game library is sourced from RAWG, which covers the vast majority of released games. If a game is missing it's usually because it's very new, very niche, or not yet indexed by RAWG.

You can submit missing games directly to RAWG.io — once added there it will appear in Achievely automatically.

This can happen when searching by name if the slug (URL-friendly name) doesn't match exactly. Try searching from the Library page instead — the search results link directly to the correct game slug so navigation will always work.

Steam privacy changes can take a few minutes to propagate to the API. Wait 2–3 minutes after changing your settings, then try refreshing the Profile page or re-entering your Steam ID.

Also double-check that Game details is set to Public separately — the profile and game details have independent visibility settings in Steam.

Game covers are served from Steam's and RAWG's CDNs. Occasional loading failures are usually caused by a slow connection or CDN hiccup — a page refresh fixes it in most cases.

If images are consistently missing, check that your browser or network isn't blocking requests to *.steamstatic.com or media.rawg.io (some ad-blockers do this).
